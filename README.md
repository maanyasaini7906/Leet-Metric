# LeetMetric 🚀

A web application that allows users to enter their LeetCode username and view their problem-solving statistics in a clean and interactive interface.

## 🔥 Features
- Fetches LeetCode user data using API
- Displays Easy, Medium, and Hard problem stats
- Circular progress visualization (pie-style)
- Simple and responsive UI

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript

## 🚀 How to Use
1. Enter your LeetCode username
2. Click on "Search"
3. View your problem-solving stats instantly
